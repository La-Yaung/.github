## What We Build

We build interactive learning tools that help Myanmar students prepare for and pass their exams.

LaYaung is a mobile-first learning platform serving students preparing for **IGCSE (Cambridge)**, **GED (US equivalency)** and the **Myanmar national curriculum (Grade 10–12)**. We replace passive video lessons with hands-on, bilingual activities — built specifically for Myanmar students, in both Burmese and English.

---

## Why We Exist

Myanmar students preparing for internationally recognised exams face a broken landscape:

- All quality resources are in English only
- No existing platform maps learning to real Cambridge or GED syllabus outcomes
- Students have no way to know if they are actually exam-ready
- Human teacher feedback on written work is inaccessible digitally

LaYaung is the first interactive, Myanmar-language platform to address all four.

## The Platform

| Feature | Detail |
|---|---|
| 📱 Mobile App | iOS & Android (React Native + Expo) + Web (Next.js) |
| ⚡ Activity Types | 9 H5P-inspired interactive types — MCQ, drag-the-words, flashcards, fill-in-blank, ordering, true/false, drag-match, mark-the-words, math input |
| 🗺 Syllabus Mapping | Every lesson mapped to IGCSE outcome codes and GED subject standards |
| 📝 Teacher Feedback | Students submit written work — teachers review directly in-app |
| 📊 Gap Analysis | Assessments surface weak syllabus areas and recommend targeted lessons |
